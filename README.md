# Research PDF Q&A

Upload up to 10 research PDFs, ask questions, get relevant excerpts.

**Stack:** PyMuPDF · MiniLM-L6-v2 · FAISS · Streamlit

## Run locally
pip install -r requirements.txt
streamlit run app.py

## Deploy
Push to GitHub → [share.streamlit.io](https://share.streamlit.io) → point to this repo.
